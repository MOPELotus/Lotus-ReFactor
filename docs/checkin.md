# 签到与调度

## 用户指令

- `#注册自动签到`
- `#注册自动签到2`
- `#补签`
- `#补签2`
- `#我的签到时间`
- `#开启签到通知`
- `#关闭签到通知`
- `#启用社区签到2`
- `#关闭社区签到2`
- `#启用原神签到2`
- `#关闭原神签到2`
- `#启用全部游戏签到2`
- `#关闭全部游戏签到2`

所有操作都允许在群聊执行。通知默认开启，私聊优先；若无法私聊，会在共同群聊 at 用户。

## 支持范围

游戏签到按上游能力支持：

- 原神
- 星铁
- 绝区零
- 崩坏2
- 崩坏3
- 未定事件簿
- 崩坏：因缘精灵

社区签到按米游社论坛支持：

- 崩坏3
- 原神
- 崩坏2
- 未定事件簿
- 大别野
- 星铁
- 绝区零
- 崩坏：因缘精灵
- 星布谷地

国际服和云游戏默认隐藏，只有用户绑定对应 token/cookie 后才参与。

## 调度

bot 主人可以设置：

- 固定模式
- 随机模式
- 计划生成时间
- 是否允许用户在固定/随机之间切换

随机模式会在前一天生成次日计划，把用户平均分布在 `00:00-23:30`。计划生成后会给用户发送签到时间图片。当天新注册的 profile 会补入当日计划，不会重新洗牌。
