# 图鉴

图鉴数据来自 `nanoka-atlas-backend`。插件侧负责触发更新、维护本地索引、注册查询指令和渲染图片。

## 初始化

首次使用时，如果本地没有：

- `data/map.json`
- `data/items`
- `gallery`

会调用后端执行全量抓取。后续定期检查版本号，只有版本变化才跑增量更新。

## 指令

- `#图鉴 星见雅`
- `#星见雅`
- `#雾切之回光`
- `#雾切`
- `#冰封迷途的勇士`
- `#本期幻想`
- `#下期幽境危战`
- `*上期末日`
- `*异相`
- `%下期防卫战`
- `#成就`
- `#成就目录2`
- `#成就查漏2`
- `#成就导入2`
- `#尘世巡游第三辑2`
- `#成就尘世巡游第三辑2`
- `#检查图鉴更新`
- `#更新图鉴`
- `#全量更新图鉴`

查询优先级按常用内容排序：角色、武器、挑战、套装、敌人、材料、物品。

裸挑战入口不会被图鉴抢占，例如 `#深渊`、`#幻想`、`#危战`、`*混沌`、`*末日`、`*异相`、`%防卫战` 等仍留给用户挑战数据查询。带“上期/本期/当期/下期”或日期语义时，才按图鉴挑战数据查询。

成就数据使用本地图鉴的所有原神成就分类。`#成就`、`#成就目录`、`#成就查漏` 展示目录和各分类完成状态；`#成就导入` 只接收椰羊导出的 JSON；分类查询支持省略中点号，例如 `#尘世巡游第三辑` 可以匹配 `尘世巡游·第三辑`。分类详情会把未完成条目前置，同名多阶段成就按整组排序；内容过长时自动分页，按顺序发送或合并转发。

## 渲染

角色展示满级基础数值、技能说明、等级数值、命座/星魂/影画、材料。

武器/光锥/音擎展示基础数值、效果、精炼/叠影/星级差异；仅数值不同的效果会合并成高亮数值列。

挑战展示周期、环境 buff、可选 buff、关卡、上下半或节点敌人和怪物图标。幽境危战图鉴只保留 N5/N6，怪物说明会清理富文本标记后展示。
