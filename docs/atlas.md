# 图鉴

图鉴数据来自 `nanoka-atlas-backend`。插件侧负责触发更新、维护本地索引、注册查询指令和渲染图片。

## 初始化

首次使用时，如果本地没有：

- `data/map.json`
- `data/items`
- `gallery`

会调用后端执行全量抓取。后续定期检查版本号，只有版本变化才跑增量更新。

## 指令

- `#图鉴 星见雅`
- `#星见雅`
- `#雾切之回光`
- `#雾切`
- `#冰封迷途的勇士`
- `#本期幻想`
- `*上期末日`
- `%下期防卫战`
- `#检查图鉴更新`
- `#更新图鉴`
- `#全量更新图鉴`

查询优先级按常用内容排序：角色、武器、挑战、套装、敌人、材料、物品。

裸挑战入口不会被图鉴抢占，例如 `#深渊`、`#幻想`、`*混沌`、`*末日`、`%防卫战` 等仍留给用户挑战数据查询。

## 渲染

角色展示满级基础数值、技能说明、等级数值、命座/星魂/影画、材料。

武器/光锥/音擎展示基础数值、效果、精炼/叠影/星级差异；仅数值不同的效果会合并成高亮数值列。

挑战展示周期、环境 buff、可选 buff、关卡、上下半或节点敌人和怪物图标。
