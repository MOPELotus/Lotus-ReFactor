# 自动签到

自动签到按 profile 执行。签到前会刷新该 profile 的登录态；结果会通知对应用户，私聊不可用时尝试共同群聊 at。

用户指令：

- `#注册自动签到2`
- `#注册本群签到2`
- `#测试签到2`
- `#开始签到2`
- `#手动签到2`
- `#补签2`
- `#开启签到通知2`
- `#关闭签到通知2`
- `#绑定通知群2`
- `#设置通知私聊2`
- `#设置通知群聊2`

游戏签到开关：

- `#启用原神签到2`
- `#关闭星铁签到2`
- `#启用绝区零签到2`
- `#启用全部游戏签到2`
- `#关闭全部游戏签到2`

社区签到开关：

- `#启用社区签到2`
- `#关闭社区签到2`

国际服和云游戏只有在绑定对应 cookie/token 后才参与签到。
