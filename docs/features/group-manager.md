# 群管理-成员导出与退群清理

返回：[上一级](../README.md) / [文档目录](../README.md) / [小功能索引](README.md)

## 功能特性

- 支持导出当前群或指定群的成员列表。
- 导出结果为 CSV 文件，适合用 VSCode、Notepad++ 或表格软件查看。
- 退群清理默认只做 dry-run 预览和审计。
- 是否实际删除配置由全局群清理策略控制。

## 指令用法

```text
#荷花群成员 [群号]
#群清理退群 <用户ID> [群号]
```

## 变量说明

- `群号`：可选，目标群号；省略时使用当前群。
- `用户ID`：必填，要清理的用户 QQ。
