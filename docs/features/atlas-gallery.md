# 图鉴查询

图鉴数据来自 `nanoka-atlas-backend`。插件侧负责触发更新、维护本地索引、注册查询指令和渲染图片。

数据更新：

- `#图鉴状态`
- `#更新图鉴`
- `#检查图鉴更新`
- `#全量更新图鉴`
- `#图鉴帮助`

常用查询：

- `#图鉴 星见雅`
- `#星见雅`
- `#雾切之回光`
- `#冰封迷途的勇士`
- `#星见雅天赋`
- `*黄泉星魂`
- `%仪玄影画`

查询优先级按常用内容排序：角色、武器、挑战、套装、敌人、材料、物品。

首次缺少 `data/map.json`、`data/items` 或 `gallery` 时会执行全量抓取。后续只在版本变化时执行增量更新。
