# 签到调度

签到调度支持固定模式和随机模式。

固定模式使用全局时间；随机模式会在前一天生成次日计划，把用户平均分布在 `00:00-23:30`，并给每个用户发送次日签到时间。

管理指令：

- `#生成签到计划`
- `#我的签到时间`
- `#执行到期签到`
- `#签到随机模式`
- `#签到固定模式 04:30`
- `#签到计划生成 00:00`
- `#签到名单列表`
- `#自动签到日志`
- `#批量刷新签到`

用户级时间设置：

- `#随机签到时间2`
- `#固定签到时间2 04:30`
- `#跟随全局签到时间2`

锅巴面板和配置文件使用七段 cron，例如 `0 0 0 * * ? *`。
