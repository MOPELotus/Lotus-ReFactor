# 签到调度-随机与固定

返回：[上一级](../checkin.md) / [文档目录](../README.md) / [小功能索引](README.md)

## 功能特性

- 固定模式使用全局签到时间。
- 随机模式会在前一天生成次日计划，并把用户平均分布在 `00:00-23:30`。
- 已生成的计划会在重启后继续使用，不会重复洗牌。
- 锅巴面板和配置文件使用七段 cron。

## 指令用法

```text
#生成签到计划
#我的签到时间
#执行到期签到
#签到随机模式
#签到固定模式 <时间>
#签到计划生成 <时间>
#签到名单列表
#自动签到日志
#批量刷新签到

#随机签到时间[profile]
#固定签到时间[profile] <时间>
#跟随全局签到时间[profile]
```

## 变量说明

- `profile`：可选，Lotus 内部 profile 序号，范围 `1..255`；省略时使用 profile 1。
- `时间`：必填，使用 `HH:mm`，例如 `04:30`。
