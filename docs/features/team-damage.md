# 队伍伤害

队伍伤害是 profile-aware 的小程序数据能力，支持原神和星铁。

原神：

- `#队伍伤害 神鹤万心`
- `#队伍伤害2 宵宫 夜兰 茜特菈莉 希诺宁`
- `#队伍伤害详情2 宵宫夜兰茜特菈莉希诺宁`

星铁：

- `*队伍伤害 迷迷遐风`
- `*队伍伤害2 银狼lv999 藿藿 风堇 小伊卡`
- `#星铁队伍伤害过程2 银狼 藿藿 风堇 小伊卡`

角色名可以空格分隔。原神会尝试识别连续拼接的角色名；星铁会优先走别名和加强版角色映射，例如银狼、藿藿这类已有 Pro 面板数据的角色会按实际面板名查询。

原神结果包含总伤害、DPS、伤害构成和角色条图。星铁结果额外展示行动记录。
