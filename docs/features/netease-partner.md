# 网易云合伙人-自动任务

返回：[上一级](../README.md) / [文档目录](../README.md) / [小功能索引](README.md)

## 功能特性

- 使用网易云音乐 App 扫码登录，并把账号 cookie 保存到运行时数据目录。
- 自动任务按配置 cron 执行，启动时会检查是否需要补跑。
- 每个任务子项独立计算随机评分，不复用总分。
- 执行完成后渲染图片报告，展示账号、任务数量和子项结果。

## 指令用法

```text
#合伙人登录
#合伙人测试
#合伙人日志
```

## 变量说明

这些指令没有额外变量。登录时按机器人提示完成扫码即可。
