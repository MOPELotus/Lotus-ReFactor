# 挑战查询-图鉴期数

返回：[上一级](../daily-note.md) / [文档目录](../README.md) / [小功能索引](README.md)

## 功能特性

- 带 `上期`、`本期`、`当期`、`下期` 或日期语义时，查询本地图鉴里的挑战期数。
- 不带期数的个人战绩入口交给 profile 查询页处理。
- 星铁图鉴挑战包含混沌回忆、虚构叙事、末日幻影和异相仲裁。
- 幽境危战图鉴只展示 N5/N6，并合并相同怪物信息。

## 指令用法

```text
#<期数><原神挑战>
*<期数><星铁挑战>
%<期数><绝区零挑战>
```

## 变量说明

- `期数`：可选，支持 `上期`、`本期`、`当期`、`下期` 或可识别日期；省略时会优先按个人战绩查询处理。
- `原神挑战`：必填，支持 `深渊`、`幻想`、`幽境危战`、`危战`。
- `星铁挑战`：必填，支持 `混沌`、`虚构`、`末日`、`异相`。
- `绝区零挑战`：必填，支持 `防卫战`、`危局`、`临界` 等图鉴已收录的挑战。
