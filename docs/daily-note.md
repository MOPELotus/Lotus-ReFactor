# 体力、面板、抽卡

## 体力

- `#全部体力`
- `#全部体力2`
- `#体力2`
- `#树脂2`
- `#便笺2`
- `*体力2`
- `%体力2`
- `#星铁体力2`
- `#绝区零体力2`

`#全部体力` 会收集当前 QQ 所有 profile、所有游戏、所有 UID。单个 UID 失败不会中断其他 UID。

星铁派遣按官方新字段显示，不再使用旧派遣字段。

## 面板

- `#更新面板2`
- `#星铁更新面板2`
- `*更新面板2`
- `#绝区零更新面板2`
- `%更新面板2`
- `#火神面板2`
- `*狼尊面板3`
- `%练度统计2`

按 profile 选择登录态和 UID，再构造一次性事件上下文转交 miao/genshin/ZZZ 底层逻辑。角色面板、角色列表、练度统计、原神深渊/幻想/幽境、星铁混沌/虚构/末日/异相、绝区零防卫战/危局/临界/月报等常用个人查询都支持 profile 后缀。

## 抽卡记录

- `#更新抽卡记录2`
- `*更新抽卡记录2`
- `#星铁更新抽卡记录2`
- `%更新抽卡记录2`
- `#绝区零刷新抽卡链接2`
- `#更新全部抽卡记录`

原神/星铁通过 profile 的 stoken 生成 authkey。绝区零优先 CK 直刷，只有获取抽卡链接时才走 authkey。

## 队伍伤害

- `#队伍伤害 神鹤万心`
- `#队伍伤害2 宵宫 夜兰 茜特菈莉 希诺宁`
- `#队伍伤害详情2 宵宫夜兰茜特菈莉希诺宁`
- `*队伍伤害2 银狼lv999 藿藿 风堇 小伊卡`
- `#星铁队伍伤害过程2 银狼 藿藿 风堇 小伊卡`

原神角色名可以空格分隔，也会尝试识别连续拼接的角色名。星铁会优先走别名和加强版角色映射，例如银狼、藿藿这类已有 Pro 面板数据的角色会按实际面板名查询。
