# 体力、面板、抽卡

## 体力

- `#全部体力`
- `#全部体力2`
- `#体力2`
- `*体力2`
- `%体力2`
- `#星铁体力2`
- `#绝区零体力2`

`#全部体力` 会收集当前 QQ 所有 profile、所有游戏、所有 UID。单个 UID 失败不会中断其他 UID。

星铁派遣按官方新字段显示，不再使用旧派遣字段。

## 面板

- `#更新面板2`
- `*更新面板2`

按 profile 选择登录态和 UID，再构造一次性事件上下文转交 miao/genshin 底层逻辑。

## 抽卡记录

- `#更新抽卡记录2`
- `*更新抽卡记录2`
- `%更新抽卡记录2`
- `#更新全部抽卡记录`

原神/星铁通过 profile 的 stoken 生成 authkey。绝区零优先 CK 直刷，只有获取抽卡链接时才走 authkey。
